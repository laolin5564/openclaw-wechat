# 微信 iPad 协议服务

## 下载可执行文件

从 [GitHub Releases](https://github.com/你的用户名/openclaw-wechat/releases) 下载对应平台的可执行文件：

- `wechat-service-darwin-arm64` - macOS Apple Silicon
- `wechat-service-darwin-amd64` - macOS Intel (待编译)
- `wechat-service-linux-amd64` - Linux x64 (待编译)
- `wechat-service-windows-amd64.exe` - Windows x64 (待编译)

下载后放到 `wechat-service/` 目录，并添加可执行权限：

```bash
chmod +x wechat-service-darwin-arm64
```
